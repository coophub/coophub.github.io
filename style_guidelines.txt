Overall Look and Feel of Roomie
===============================

The website will feature a main page and only a main page which will contain everything the user would need to know
about. This approach allows for maximum engagement as users would only have to scroll and would not have to wait for the
page to refresh.

Colours
========
Coophub should look professional and feel like a workplace.

- #222222 (variant of some sort)
- #eeeeee (variant of some sort)
